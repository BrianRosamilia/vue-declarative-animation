Class list

  • .(x)-(y)

Description

The position preset classes are to be used on an element whose position value is something other than static.

Values

The (x) values that can be plugged in are: left, center, right.

The (y) values that can be plugged in are: top, center, bottom.

Example

<div class="h-1-2-vh relative primary r">
    <figure class="square danger absolute left-top"></figure>
    <figure class="square danger absolute left-center"></figure>
    <figure class="square danger absolute left-bottom"></figure>
    <figure class="square danger absolute center-top"></figure>
    <figure class="square danger absolute center-center"></figure>
    <figure class="square danger absolute center-bottom"></figure>
    <figure class="square danger absolute right-top"></figure>
    <figure class="square danger absolute right-center"></figure>
    <figure class="square danger absolute right-bottom"></figure>
</div>
