Description

Consider using the .pagination component when you want to break up things such as tables or sections of content into "pages".

Example

  • previous
  • 1
  • 2
  • 3
  • next 
<ul class="pagination">
    <li class="page">
        <a href="" class="page-a">previous</a>
    </li>
    <li class="page">
        <a href="" class="page-a">1</a>
    </li>
    <li class="page">
        <a href="" class="page-a">2</a>
    </li>
    <li class="page is-active">
        <a href="" class="page-a">3</a>
    </li>
    <li class="page is-disabled">
        <a href="" class="page-a">next</a>
    </li>
</ul>
