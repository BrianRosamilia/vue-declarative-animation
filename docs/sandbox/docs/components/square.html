Description

The figure.square is a versatile element that can be used to contain an img, or some text. By default, the dimensions of a figure.square are 1:1 with the font-size that is applied to it, with a default font-size of 1.6rem.

Example

<figure class="square dark"></figure>

Multipliers

Use the multiplier classes on a figure.square to change the size ratio in relation to whatever font-size is applied to the square.

<figure class="square dark"></figure>
<figure class="square dark x2"></figure>
<figure class="square dark x3"></figure>
<figure class="square dark x4"></figure>

Text

Use the .text class on a figure.square to center the text within, along the x and y axes.

<figure class="square dark x4 text">SB</figure>

Image

Any img element used within a figure.square will take up the full dimensions of the square.

<figure class="square x4">
    <img src="/path/to/img" alt="image">
</figure>

Font size scaling

Use the font-size utility classes on a figure.square to change the dimensions of the figure.square.

<figure class="square dark h5"></figure>
<figure class="square dark"></figure>
<figure class="square dark h3"></figure>
<figure class="square dark h2"></figure>
<figure class="square dark h1"></figure>
