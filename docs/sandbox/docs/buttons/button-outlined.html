Class list

  • .button.outlined

Description

Use the .outlined class on a .button to have outline styling applied.

Example

<button class="button outlined">button</button>

Theme classes

The theme classes Sandbox provides set some extra pseudo class styling specifically for outlined buttons.

<button class="button primary outlined">primary</button>
<button class="button info outlined">info</button>
<button class="button caution outlined">caution</button>
<button class="button danger outlined">danger</button>
<button class="button light outlined">light</button>
<button class="button dark outlined">dark</button>
